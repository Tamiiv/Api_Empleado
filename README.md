# ASP.NET CORE WEB API

Desarrollo CRUD de Empleado,

## BASE DE DATOS

```bash
CREATE DATABASE DBEmpleado

USE DBEmpleado

CREATE TABLE Empleado(
IdEmpleado INT PRIMARY KEY IDENTITY,
Nombre VARCHAR(50),
Correo VARCHAR(50),
Sueldo DECIMAL(10, 2)
)
```
